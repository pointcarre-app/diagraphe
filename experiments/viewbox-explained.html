üéØ The Core Concept

viewBox defines an internal coordinate system (your "grid paper"). width/height define the rendered size (the "photocopy size").

viewBox="minX minY width height" ‚Äî The internal canvas you draw on.

Think of it like this: You draw on a 100√ó100 grid. You can then display that drawing at 50px, 200px, or 500px ‚Äî the internal coordinates never change, only the output scale.

2Ô∏è‚É£ Aspect Ratio Mismatch (viewBox vs Rendered Size)

What happens when viewBox="0 0 100 100" (square) but rendered size is rectangular? This is where preserveAspectRatio comes in.

xMidYMid meet (default)

Fits inside, maintains ratio, centers

preserveAspectRatio="xMidYMid meet"

xMidYMid slice

Fills & clips, maintains ratio, centers

preserveAspectRatio="xMidYMid slice"

none

Stretches to fill exactly

‚ö†Ô∏è The "none" option distorts the circle into an ellipse. Usually you want "meet" or "slice".

6Ô∏è‚É£ Zoom Effect (viewBox width/height)

Smaller viewBox dimensions = zoomed in. Larger viewBox dimensions = zoomed out. The rendered SVG size stays the same!

viewBox="25 25 50 50"
2x zoom (center)
viewBox="0 0 100 100"

viewBox="-100 -100 300 300"

üîç Zoom formula: zoom = renderedSize / viewBoxSize. viewBox="0 0 50 50" on a 100px SVG = 2x zoom!

üìã Recommendations for Diagraphe

For Fixed-Size Graphs

  • Use matching viewBox and width/height
  • viewBox="0 0 400 400" + width="400"
  • 1:1 mapping, no scaling surprises

For Resizable Graphs

  • Fixed viewBox as internal coordinate system
  • Variable width/height for display size
  • Use preserveAspectRatio="xMidYMid meet"

For Pan/Zoom Support

  • Keep rendered size fixed
  • Modify viewBox minX/minY for panning
  • Modify viewBox width/height for zooming

For Responsive (CSS-sized)

  • width="100%" or CSS sizing
  • Fixed viewBox for consistent coordinates
  • SVG auto-scales to container

Suggested API Pattern

new Diagraphe({
  // Internal coordinate system (fixed)
  viewBox: { width: 400, height: 400 },
  
  // OR for custom origin/zoom:
  viewBox: { minX: -10, minY: -10, width: 420, height: 420 },
  
  // Display size (can be changed)
  width: 400,    // or "100%" for responsive
  height: 400,
  
  // Aspect ratio handling
  preserveAspectRatio: "xMidYMid meet"  // default
})
